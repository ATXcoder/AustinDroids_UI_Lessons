Introduction & Overview

Hello World!

Welcome! The following series of lessons on the foundations of Android UI Development is being presented to the AustinDroids group. These lessons will be used to teach members the basics regarding Android UI and will cover subjects such as:

  • Fragments
  • Layouts
  • Custom Styles
  • and much more...
Each lesson will cover a different area of the Android UI process. A simple weather application will also be built using the concepts learned in this series.

Tools

Over the course of this series members will be introduced to a couple of different tools that are often used in the development process. These tools will range from applications for building mockups, to graphic programs, and of course programs for actually building our applications. Some of the tools we will be exploring are:

Concept Tools

Tools for helping to flush out a idea/concept

  • Freemind
  • Programmable Web (API)
Mockup Tools

Tools for creating Mockups of your Android Apps

  • Pencil
  • Balsamiq Mockups
Development Tools

Tools for development

  • Eclipse
  • Android Studio
  • Android Asset Studio
Testing Tools

Tools for testing

  • Genymotion
